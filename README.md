# Презентация скролла по оси Z с моей домашней кошькой.

---

![](/ForMD.jpg)

## _Ключевые моменты_:

Скролл по оси Z, все элементы найденные на странице добавляются в массив, что бы раздать значения которые меняются при скролле по оси Z.
в Переменную _top_ записывается расстояние сверху до текущего фрейма.
Параметр _delta_ вычисляет последнюю позицию за пределом функции скролла _delta = lastPos- top_.
в Массив _zVals_ пушим обновленные значения _i \* zSpacing (- 1000)_ для того что бы раздать каждому элементу значения по убывающей. + текущий _zSpacing_ для расстояния.
_zVals+delta \* -5_ ( -5 это скорость пролистывания.)
Вся логика работы движка скролла написана на JavaScript без библиотек.
